## 프로젝트 배경

이 프로젝트는 카페의 주문시스템 활용을 위한 위한 웹 애플리케이션 제작입니다. 카페 내 부족한 키오스크 개수로 발생하는 대기시간을 줄이기 위해 개인 디바이스로 주문할 수 있도록 하고, 키오스크와 유사한 기능을 제공함으로 자영업자들의 키오스크 설치 비용 발생 문제를 해결할 수 있습니다. 

#### 개발환경

- Back-end: Django Rest Framework
- DB: MariaDB
- Front-end: React
